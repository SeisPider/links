Volcano

  • PF3D: PF3D is a Python module that facilitates volcanic ash dispersal modelling using the FALL3D package

Geodynamic

  • ASPECT: A parallel, extensible finite element code to simulate convection in both 2D and 3D models.

  • PyLith: an open-source finite-element code for dynamic and quasistatic simulations of crustal deformation, primarily earthquakes and volcanoes.

  • Rayleigh: a 3-D convection code designed for the study of dynamo behavior in spherical geometry.

  • citcoms: a finite element code designed to solve compressible thermochemical convection problems relevant to Earth s mantle.

Seismological forward modelling codesBack to Top

Spectrum element method

  • SES3D: SES3D is a program package for the simulation of elastic wave propagation and waveform inversion in a spherical section. The package is based on a spectral-element discretisation of the seismic wave equation combined with adjoint techniques.


Reflectivity method

  • fk: Frequency-Wavenumber (FK) synthetic seismogram package

  • telewavesim: Teleseismic body wave simulation with propagation matrix method

Modal summation

  • CPS330: General computer program in seismology

Ray theory

  • RAYSUM: Ray-theoretical modelling of teleseismic waves in dipping, anisotropic structures

  • aser1.0: Generalized Ray Theory (GRT) synthetic seismogram package

Compound method

  • Multitel3: Combination of classical ray theory and haskell propagation metrix method

  • GEMINI: Calculate seismogram of teleseismic events

Seismological inversion codesBack to Top

Structure

Full Waveform

  • lasif: a data-driven end-to-end workflow tool to perform adjoint full seismic waveform inversions.

  • SeisFlow: differs from previous open-source software in emphasizing both flexibility and HPC portability.

  • DSM kernel: calculate 3D finite frequency Fréchet sensitivity kernels (or 3D waveform partial derivatives) for 1D reference Earth models using Direct Solution Method.

Surface wave

  • topo_sp_cu: ray theoretic surface wave tomography code.

  • FMTOMO: The forward problem of traveltime prediction is solved using a multi-stage fast marching method (FMM), which is a sophisticated grid-based eikonal solver.

SKS

  • splitlab: a Matlab based GUI to measure the splitting of (teleseismic) shear waves.

  • Pytheas: software for local shear-wave splitting analysis

Receiever function

  • rf: software for local shear-wave splitting analysis

  • hk: Receiver function package (deconvolution and H-k stacking)

  • ccp: Common-Conversion-Point (CCP) stacking of receiver function

Source

Moment tensor

  • MTfit: a Bayesian forward model inversion code for moment tensor and double-couple source inversion using different data types, based on the Bayesian approach presented in Pugh et al, 2016a and Pugh, 2015.
